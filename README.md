# Simple Web Server Implemented in C

## Environment

* Ubuntu 16.04.2 LTS

## Build

```
$ make [debug=-g] (optional)
$ ./server [port]
```

## Reference

* [C-Thread-Pool](https://github.com/Pithikos/C-Thread-Pool)
* [http-parser](https://github.com/nodejs/http-parser)
